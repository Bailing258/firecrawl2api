# Firecrawl Search + Map

优先使用：
- `search_web`
- `map_site`

## 什么时候用 search_web

- 用户问“搜一下”
- 需要按主题找网页
- 需要快速找到目标页面 URL 集合

## 什么时候用 map_site

- 用户已给定域名
- 需要列出站点页面
- 需要做站点入口、目录、链接清单

## 输出建议

- Search：返回最相关结果、标题、URL、简要摘要
- Map：返回重要路径分组，而不是无脑贴所有链接
